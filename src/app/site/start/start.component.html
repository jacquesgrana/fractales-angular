Accueil

Ceci est un application qui dessine des fractales de Julia.

Il est possible des modifier :

Le nombre de départ [Seed] (nombre complexe dont on peut modifier la partie réelle et la partie imaginaire) : valeur de départ de la suite appliquée dans la boucle de calcul.

Le module limite [Limite] : détermine si le pixel correspondant au module du nombre complexe testé est affichable.

Le nombre maximum d'itérations [Nombre maximum d'itérations] : détermine le nombre maximum de boucles de la boucle de calcul.

Une liste de fractales pré-enregistrée est accessible dans les réglages.

Celle application a été codée en Typescript/Html/Scss sous Angular en utilisant Material et BootStrap.

Une petite partie du code a été trouvée sur internet (algorithme de la boucle de calcul de la fractale et la bibliothèque pour le calcul matriciel).
